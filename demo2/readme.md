## 软件架构
rollup + less + jquery + jquery.tmpl.min.js

## 构建 Start

1. `yarn`
2. `yarn start` 项目启动
3. `yarn build` 项目打包

## 项目理解
1. src是需要编写代码的地方
2. example/index.html  项目启动html
